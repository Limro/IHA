\section{USB}

Universal Serial Bus.

	\begin{itemize}
	\item Plug \& play
	\item Den er Isochronous (sender med en bestemt rate)
	\item Asynchronous (Kan sende og modtage)
	\item Efter USB 2.0 kom OTG (On The Go)
	\item Max 5 meter uden ekstra power
	\item Bruger NRZI coding (hvis et signal skifter det 0, hvis ikke - 1)
	\item Micro B og Mini B er standard sammen med Standard A
	\item Alle enheder har VID/PID, som skal købes (når man ikke udvikler og tester)
	\item Går i dvale efter 3.0 ms uden aktivitet og yderligere 7.0 ms for at lukke ned.
	\item Der sendes kun 500 micro-A ved dvale
	\end{itemize}
Overførelse - handshake:
	\begin{enumerate}
	\item Sync
	\item PID
	\item ADDR
	\item ENDP
	\item CSC8
	\item EOP
	\end{enumerate}
Overførelse - data:

	\begin{enumerate}
	\item Sync
	\item PID
	\item DATA
	\item CSC16
	\item EOP
	\end{enumerate}
Generelt:

	\begin{itemize}
	\item LSB sendes først, MSB sidst
	\item Alle pakker starter med en \textit{sync} (8 cykler for normal og 16 for high speed)
	\item Der skal kendes en adresse at sende til
	\item Der skal være et Endpoint
	\item 
	\end{itemize}
USB ENdpoints:

	\begin{itemize}
	\item Skal have Endpoint0, brug til transport control
	\item Kan have op til 31 endpoints per enhed
	\item Unidirectional - enten hente, eller modtage
	\end{itemize}
